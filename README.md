# Angular AOT Dynamic Components

Simple project for creating Angular Dynamic Components with The Ahead-of-Time (AOT) compiler.


## Installation
```sh
npm i
```


## Run app local
In two separate terminals

```sh
npm start
```

and
```sh
npm run json-server
```

## Run the app on SSR mode 

In two separate terminals

```sh
npm run start:ssr
```

and
```sh
npm run json-server
```
